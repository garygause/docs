## Basic Formatting

_worids in italics_ = \_words in italics\_

or

_italicized text_ = \*italicized text\*

**bold text** = \*\*bold text\*\*

> blockquote = \> blockquote

> Mary had a little lamb.
>
> Little lamb
>
> Little lamb

code = \`code`

`const a = 1;
const b = 2;
const result = a + b;`

fenced code blocks = \`\`\` code \`\`\`

```
const a = 1;
const b = 2;
const result = a + b;
```
